grinway/installer
========

## Configuration

:hammer: Configure the behaviour of `grinway/installer`\
with `%kernel.project_dir%/composer.json` in `extra` section

The following example contains default values:
```json
    "extra": {
        "grin_way_installer": {
            "enable": true,
            "enable_copy_bundles_configs_to_project_dir": true
        }
    },
```
